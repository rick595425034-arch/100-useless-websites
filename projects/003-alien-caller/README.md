# 003 · 深空文明呼叫终端

“挑战 100 个整活网站”的第三个作品。它把自己包装成一套非常严肃的深空通信系统：扫描、握手、翻译、建立链路，然后把一次普通的拒接交给大模型处理。

这是一个面向 10–30 秒短视频表演设计的 macOS 桌面应用。关键中文使用大字号，来电、翻译、远端反应与桌面接管按时间线连续演出。

## 运行要求

- macOS 10.15 或更高版本
- Node.js 22
- Xcode Command Line Tools
- Apple Silicon 或 Intel Mac

## 本地运行

```bash
npm ci
npm run build:screen-access
npm start
```

原生屏幕权限模块会在本机从 [`electron/screen-access.c`](./electron/screen-access.c) 编译生成。生成的 `.node` 文件不会提交到仓库。

## 测试

```bash
npm test
```

测试会保护主要演出节点、真实应用图标捕获、无缝拒接衔接、紧急退出方式，以及“不接触用户文件”的安全边界。

## 构建 DMG

```bash
npm run dist:mac
```

生成的应用与 DMG 位于 `release/`，该目录不会进入 Git。打包脚本会优先使用本机可用的 Apple 开发签名身份；没有签名环境时仍可阅读和测试源码。

## 屏幕权限

为了让 UFO 演出使用当前桌面与真实应用图标，需要在：

`系统设置 → 隐私与安全性 → 屏幕与系统音频录制`

允许“深空文明呼叫终端”，然后重新打开应用。拒接后，理由界面会立即覆盖屏幕，桌面快照在遮罩下方于本机完成，不会先暴露一段未处理的桌面画面。

## 安全说明

- 不移动、删除、修改或上传任何本地文件。
- 所有“资料被带走”和“设备锁定”均为纯视觉演出。
- 桌面快照仅存在于当前应用内存，演出结束后立即释放。
- 全屏演出期间随时按 `Esc` 恢复桌面。
- `Command + Q` 也可退出应用。

## 目录结构

```text
assets/    UFO 视觉素材
build/     应用图标等打包资源
electron/  Electron 主进程、预加载脚本与原生权限源码
scripts/   原生模块编译与 macOS 签名脚本
src/       主界面、来电界面与桌面接管演出
tests/     安全边界与剧情流程测试
```
